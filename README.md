## 👋 About Me

I'm an AI & Software Engineer focused on building **intelligent, production-grade systems** — from computer vision pipelines to LLM-powered agents and automated workflows. I stay deliberately close to the fast-moving edge of AI: agentic architectures, retrieval-augmented generation, model orchestration, and the automation tooling that turns AI capability into real business value.

I care about **depth over trend-chasing** — understanding how these systems work under the hood (embeddings, context management, tool-use protocols, fine-tuning) so I can build things that are reliable, not just impressive in a demo.
